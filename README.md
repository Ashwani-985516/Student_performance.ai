# Student Performance AI 🎓

A full-stack AI-powered system designed to analyze, predict, and improve student academic performance using machine learning and data analytics.

## 🌟 Features

- **AI-Powered Analytics**: Machine learning models for performance prediction
- **Data Visualization**: Interactive dashboards and charts
- **Student Insights**: Detailed performance analysis and recommendations
- **Full-Stack Architecture**: Modern frontend and backend integration
- **Scalable Design**: Built for growth and extensibility

## 📊 Project Composition

- **Jupyter Notebooks** (83.1%): Data analysis, ML models, and experimentation
- **JavaScript** (14.7%): Frontend interactive components
- **CSS** (1.8%): Styling and UI design

## 🛠️ Tech Stack

### Backend
- Python
- Jupyter Notebook for data analysis
- Machine Learning frameworks (scikit-learn, TensorFlow, etc.)

### Frontend
- JavaScript
- CSS for styling
- Interactive UI components

## 🚀 Getting Started

### Prerequisites
- Python 3.7+
- Jupyter Notebook
- Node.js (for frontend dependencies)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Ashwani-985516/Student_performance.ai.git
   cd Student_performance.ai
   ```

2. **Set up Python environment**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```

3. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

4. **Install JavaScript dependencies** (if applicable)
   ```bash
   npm install
   ```

## 📚 Usage

1. Open Jupyter notebooks to explore data analysis and ML models
